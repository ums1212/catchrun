# catchrun

run and catch

### 매치 만들기와 참가하기
1. 게임 만들기
2. 게임 찾기(목록 찾기, 번호 찾기)
3. 게임 참가는 초대코드(비밀번호)가 있어야 함.
4. 경찰과 도둑 인원수 및 비율 정하기
### 플레이 방식
5. 경찰은 도둑을 잡으면 도둑의 QR을 촬영 또는 NFC 태그를 해야 함. 잡힌 도둑은 자유 상태에서 수감 상태로 바뀜
6. 감옥에는 수감상태인 도둑들이 모여있고 잡히지 않은 도둑이 수감상태인 도둑의 QR 또는 NFC 태그를 통해 한 명씩 자유 상태로 바꿀 수 있음
7. 감옥 열쇠 NFC카드를 태그하면 사용한 도둑은 수감 상태에서 자유 상태로 바뀌고 5분동안 다른 수감 도둑을 풀어줄 수 없음. 경찰에게 감옥 열쇠 사용 알림이 감. 감옥 열쇠는 단 한 번만 사용할 수 있음.
8. 게임은 제한시간을 설정할 수 있고 제한시간 동안 경찰이 도둑들을 모두 잡거나 도둑들이 잡히지 않아야 함.
### 점수 산정
9. 경찰은 도둑을 잡으면 점수를 얻음.
10. 도둑은 일정시간이 지날때마다 생존점수를 받음. 수감 도둑을 풀어줬을 때 보너스 점수 받음.
11. 아이템을 사용할 때 점수가 차감됨.
12. 게임이 끝나면 경찰과 도둑 진영의 승리를 보여주고 각 참가자의 점수를 순위표로 보여줌.
### 추후 추가 기능
12. 경찰의 스캐너 기능(근처의 도둑 위치를 알려줌. 일회성)
